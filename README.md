---

## 📖 How It Works

1. Enter any text or URL.
2. Click **Generate QR Code**.
3. The application sends your input to the QR Code API.
4. A QR code is generated instantly.
5. Click **Regenerate** to create another QR code.

---

## 🌐 API Used

QR Code Generator API

```
https://api.qrserver.com/v1/create-qr-code/
```
